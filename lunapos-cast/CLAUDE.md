# Luna Pos Cast（キャスト向け画面）

> 共通ルール・DB スキーマ・ブランドカラーは親 `../CLAUDE.md` を参照。

## 概要

| 項目 | 内容 |
|------|------|
| 目的 | キャスト本人がシフト・売上・指名・給与を確認 |
| ユーザー | キャスト（ホステス） |
| 端末 | スマートフォンのブラウザ |
| 技術スタック | Vite + React + TypeScript |
| 状態 | **セットアップのみ**（個人モードを優先開発予定） |

## 機能（予定）

### コア機能（店舗連携モード）
- 自分のシフト確認・シフト希望提出
- 本日の出退勤状況確認
- 自分の売上・指名数の確認
- 給与明細の確認
- プロフィール編集
- 操作ログ（自分の関連分）: 給与計算の根拠を自分で確認できる

### 個人モード（PLG の入口・最優先）

店舗が Luna Pos を導入していなくても、キャスト個人が無料で使い始められるモード。

- 自分の売上・指名数・バック額を手動で記録できる「個人メモ」機能
- 店舗の POS 連携なしで単独動作
- オーナー許可不要・完全無料 → 導入ハードルゼロ
- 個人モードで蓄積したデータは、店舗が正式導入した際にそのまま移行・連携

**狙い**: キャストが個人利用 → 「店全体で入れたらもっと便利」→ オーナーに提案 → 店舗導入

## 注意事項

- キャスト本人は出退勤の打刻はできない（POS端末からのみ）
- 閲覧は自分のデータのみに制限（RLSで制御）
- DB テーブルの R/RW 権限は親 CLAUDE.md 参照
