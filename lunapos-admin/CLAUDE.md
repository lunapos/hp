# Luna Pos Admin（オーナー管理画面）

> 共通ルール・DB スキーマ・ブランドカラーは親 `../CLAUDE.md` を参照。

## 概要

| 項目 | 内容 |
|------|------|
| 目的 | 売上分析・メニュー設定・給与計算・顧客管理 |
| ユーザー | 店舗オーナー・マネージャー |
| 端末 | PC・タブレットのブラウザ |
| 技術スタック | Vite + React + TypeScript |
| 状態 | **セットアップのみ**（Floor Phase 2 以降に本格開発） |

## 機能（予定）

### コア機能
- 売上分析・レポート（日次・月次・期間指定）
- メニュー管理（追加・編集・価格変更・カテゴリ管理）
- キャスト管理（登録・編集・シフト管理）
- 給与計算
- 顧客管理（来店履歴・顧客ランク・ボトルキープ）
- 店舗設定（税率・サービス料率・指名料・セットプラン）
- ルーム・テーブル設定
- インボイス設定（T+13桁の登録番号 → POS レシートに自動印字）

### 将来機能
- 不正防止（操作ログ閲覧）: POS の全操作ログを閲覧・検索・フィルタ
- 異常パターンの検出・アラート

## 権限ルール

- 設定変更は管理画面が権限を持つ（POSは参照のみにする方向）
- 出退勤の「修正」は管理画面の権限
- DB テーブルの R/RW 権限は親 CLAUDE.md 参照
